# Food Expiry Date Decision Tree

## How to Use This Decision Tree

Follow the questions from top to bottom. Each answer leads you to the next question or to a final shelf life estimate. This tree helps you determine appropriate expiry dates based on the five main categories: intrinsic properties, processing, packaging, storage, and microbial factors.

---

## Master Decision Tree

```
START: What type of food are you evaluating?
│
├─► FRESH MEAT/POULTRY/SEAFOOD → Go to Tree A
├─► DAIRY PRODUCTS → Go to Tree B
├─► FRESH PRODUCE → Go to Tree C
├─► BAKED GOODS → Go to Tree D
├─► CANNED/JARRED GOODS → Go to Tree E
├─► DRIED FOODS → Go to Tree F
├─► FROZEN FOODS → Go to Tree G
└─► PROCESSED/PACKAGED FOODS → Go to Tree H
```

## Tree A: Fresh Meat/Poultry/Seafood

```
START: What type is it?
│
├─► SEAFOOD (Fish/Shellfish)
│   │
│   ├─► Is it whole fish or shellfish?
│   │   ├─► YES → Storage temp?
│   │   │   ├─► On ice (32°F) → 1-2 days
│   │   │   ├─► Refrigerated (35-38°F) → 1-2 days
│   │   │   └─► Refrigerated (40°F+) → 1 day maximum
│   │   │
│   │   └─► NO (filleted/processed)
│   │       ├─► Vacuum packed? 
│   │       │   ├─► YES → 4-7 days (refrigerated)
│   │       │   └─► NO → 1-2 days
│   │       └─► Smoked?
│   │           ├─► YES (cold smoked) → 7-14 days
│   │           └─► YES (hot smoked) → 7-10 days
│   │
├─► POULTRY (Chicken/Turkey/Duck)
│   │
│   ├─► Is it whole or parts?
│   │   ├─► WHOLE BIRD
│   │   │   ├─► Fresh → 1-2 days (32-35°F)
│   │   │   ├─► Frozen → 12 months (0°F)
│   │   │   └─► Cooked → 3-4 days refrigerated
│   │   │
│   │   └─► PARTS (breasts, thighs, wings)
│   │       ├─► Fresh → 1-2 days (32-35°F)
│   │       ├─► Vacuum packed → 2-4 days
│   │       ├─► Frozen → 9 months (0°F)
│   │       └─► Ground poultry → 1-2 days fresh, 3-4 months frozen
│   │
└─► RED MEAT (Beef/Pork/Lamb)
    │
    ├─► What cut/form?
    │   ├─► GROUND MEAT
    │   │   ├─► Fresh, standard packaging → 1-2 days (35-38°F)
    │   │   ├─► Vacuum packed → 2-4 days
    │   │   ├─► Frozen → 3-4 months (0°F)
    │   │   └─► Cooked → 3-4 days refrigerated
    │   │
    │   ├─► STEAKS/ROASTS
    │   │   ├─► Fresh, standard packaging → 3-5 days (35-38°F)
    │   │   ├─► Vacuum packed → 7-14 days
    │   │   ├─► Frozen → 6-12 months (0°F)
    │   │   └─► Cooked → 3-4 days refrigerated
    │   │
    │   └─► PROCESSED (bacon, sausage, deli meat)
    │       ├─► Unopened package?
    │       │   ├─► YES → Check "use by" date + 3-5 days
    │       │   └─► NO → 3-7 days depending on type
    │       │
    │       ├─► Cured/smoked?
    │       │   ├─► YES → 7-14 days refrigerated
    │       │   └─► NO → 3-5 days refrigerated
    │       │
    │       └─► Frozen? → 1-2 months for best quality

IMPORTANT MODIFIERS:
- Temperature abuse (left out >2 hours) → DISCARD
- Slimy texture → DISCARD
- Off odor → DISCARD
- Color changes (grey/green) → DISCARD
```

## Tree B: Dairy Products

```
START: What type of dairy?
│
├─► MILK
│   │
│   ├─► What processing?
│   │   ├─► PASTEURIZED (standard)
│   │   │   ├─► Unopened → 7-14 days past "sell by"
│   │   │   └─► Opened → 5-7 days (keep at 35-38°F)
│   │   │
│   │   ├─► ULTRA-PASTEURIZED
│   │   │   ├─► Unopened → 30-90 days refrigerated
│   │   │   └─► Opened → 7-10 days
│   │   │
│   │   └─► UHT (aseptic packaging)
│   │       ├─► Unopened → 6-9 months (room temp OK)
│   │       └─► Opened → 7-10 days (must refrigerate)
│   │
│   └─► Fat content impact?
│       ├─► Whole milk → Standard times above
│       ├─► Skim milk → +2-3 days (less fat = less rancidity)
│       └─► Buttermilk → +3-5 days (cultured = more acid)
│
├─► CHEESE
│   │
│   ├─► Moisture content?
│   │   ├─► HARD (Parmesan, aged cheddar)
│   │   │   ├─► Unopened → 6-9 months
│   │   │   ├─► Opened → 3-4 weeks (wrapped tightly)
│   │   │   └─► Grated → 3-5 days
│   │   │
│   │   ├─► SEMI-HARD (Cheddar, Swiss, Gouda)
│   │   │   ├─► Unopened → 2-4 months
│   │   │   ├─► Opened → 3-4 weeks
│   │   │   └─► Sliced → 2-3 weeks
│   │   │
│   │   ├─► SOFT (Brie, Camembert, fresh mozzarella)
│   │   │   ├─► Unopened → 1 week
│   │   │   └─► Opened → 3-5 days
│   │   │
│   │   └─► FRESH (Cottage, ricotta, cream cheese)
│   │       ├─► Unopened → 1-2 weeks
│   │       └─► Opened → 5-7 days
│   │
│   └─► Packaging enhancement?
│       ├─► Vacuum sealed → +100% to times above
│       ├─► Wax coating → +50% (hard cheese only)
│       └─► Standard wrap → Base times above
│
├─► YOGURT/CULTURED PRODUCTS
│   │
│   ├─► Unopened → 1-3 weeks past "best by"
│   ├─► Opened → 7-10 days
│   └─► Greek yogurt → +3-5 days (lower moisture)
│
├─► CREAM/SOUR CREAM
│   │
│   ├─► Heavy cream (unopened) → 1 month
│   ├─► Heavy cream (opened) → 5-7 days
│   ├─► Sour cream (unopened) → 1-3 weeks
│   └─► Sour cream (opened) → 7-10 days
│
└─► BUTTER
    │
    ├─► Salted butter (refrigerated) → 2-3 months
    ├─► Unsalted butter (refrigerated) → 1-2 months
    ├─► Salted butter (frozen) → 9-12 months
    └─► Unsalted butter (frozen) → 6-9 months

QUALITY INDICATORS:
- Sour smell → DISCARD
- Mold on soft cheese → DISCARD entire package
- Mold on hard cheese → Cut off 1 inch around mold, rest OK
- Separation → Normal, stir and use
- Curdling in milk → DISCARD
```

## Tree C: Fresh Produce

```
START: What category?
│
├─► LEAFY GREENS (lettuce, spinach, kale)
│   │
│   ├─► Pre-washed/bagged?
│   │   ├─► YES, unopened → 5-7 days
│   │   ├─► YES, opened → 3-5 days
│   │   └─► NO (whole head) → 7-10 days
│   │
│   ├─► Storage conditions?
│   │   ├─► High humidity (95%) → Maximum life
│   │   ├─► Low humidity → -50% shelf life
│   │   └─► Washed and dried → +2-3 days
│   │
│   └─► Type specific:
│       ├─► Iceberg lettuce → 7-10 days
│       ├─► Romaine → 5-7 days
│       ├─► Spinach → 5-7 days
│       └─► Arugula → 3-5 days (most delicate)
│
├─► FRUITS (Climacteric - ripen after picking)
│   │
│   ├─► ETHYLENE PRODUCERS
│   │   ├─► Bananas
│   │   │   ├─► Green → 5-7 days (room temp to ripen)
│   │   │   ├─► Ripe → 2-3 days (room temp)
│   │   │   └─► Refrigerated when ripe → +3-5 days
│   │   │
│   │   ├─► Avocados
│   │   │   ├─► Unripe → 4-5 days (room temp)
│   │   │   ├─► Ripe → 2-3 days (room temp)
│   │   │   └─► Refrigerated when ripe → +3-5 days
│   │   │
│   │   ├─► Apples
│   │   │   ├─► Room temp → 5-7 days
│   │   │   ├─► Refrigerated → 1-2 months
│   │   │   └─► Ethylene production impacts nearby items
│   │   │
│   │   └─► Tomatoes
│   │       ├─► Unripe → 3-5 days (room temp only)
│   │       ├─► Ripe → 3-5 days (room temp)
│   │       └─► Never refrigerate (texture damage)
│   │
│   └─► NON-CLIMACTERIC (don't ripen after picking)
│       ├─► Berries
│       │   ├─► Strawberries → 3-5 days (refrigerated)
│       │   ├─► Blueberries → 7-10 days (refrigerated)
│       │   ├─► Raspberries → 2-3 days (most delicate)
│       │   └─► Check daily, remove any moldy ones
│       │
│       ├─► Citrus
│       │   ├─► Room temp → 3-7 days
│       │   ├─► Refrigerated → 2-4 weeks
│       │   └─► Lower ethylene sensitivity
│       │
│       └─► Grapes
│           ├─► Refrigerated → 7-14 days
│           └─► Room temp → 2-3 days
│
└─► VEGETABLES
    │
    ├─► ROOT VEGETABLES
    │   ├─► Potatoes
    │   │   ├─► Cool (50-60°F), dark, dry → 1-2 months
    │   │   ├─► Room temp → 1-2 weeks
    │   │   └─► Never refrigerate (starch→sugar)
    │   │
    │   ├─► Onions
    │   │   ├─► Cool, dry, good air flow → 1-2 months
    │   │   └─► Cut/peeled → 7-10 days refrigerated
    │   │
    │   ├─► Carrots
    │   │   ├─► Refrigerated, high humidity → 3-4 weeks
    │   │   └─► Room temp → 3-5 days
    │   │
    │   └─► Beets/Turnips
    │       └─► Refrigerated → 2-3 weeks
    │
    ├─► CRUCIFEROUS (broccoli, cauliflower, cabbage)
    │   ├─► Broccoli → 7-14 days refrigerated
    │   ├─► Cauliflower → 1-2 weeks refrigerated
    │   └─► Cabbage → 1-2 months refrigerated
    │
    ├─► VINE VEGETABLES (cucumbers, peppers, eggplant)
    │   ├─► Cucumbers → 1 week (don't refrigerate below 50°F)
    │   ├─► Bell peppers → 1-2 weeks refrigerated
    │   └─► Eggplant → 5-7 days (cool, not cold)
    │
    └─► SQUASH
        ├─► Summer squash (zucchini) → 4-5 days
        └─► Winter squash (butternut) → 2-3 months (cool, dry)

ETHYLENE SENSITIVITY CHART:
High Sensitivity (keep away from producers):
- Leafy greens, broccoli, cucumbers, peppers

Low Sensitivity (can store together):
- Citrus, berries, root vegetables
```

## Tree D: Baked Goods

```
START: What type of baked good?
│
├─► BREAD
│   │
│   ├─► What type of bread?
│   │   ├─► Commercial sliced bread
│   │   │   ├─► Room temp → 5-7 days (with preservatives)
│   │   │   ├─► Refrigerated → 7-14 days (goes stale faster!)
│   │   │   └─► Frozen → 3-6 months
│   │   │
│   │   ├─► Artisan/homemade (no preservatives)
│   │   │   ├─► Room temp → 2-3 days
│   │   │   ├─► Refrigerated → Not recommended
│   │   │   └─► Frozen → 3-6 months
│   │   │
│   │   └─► Whole grain
│   │       ├─► Room temp → 3-5 days (oils = shorter life)
│   │       └─► Frozen → 2-3 months
│   │
│   └─► Packaging?
│       ├─► Plastic bag, sealed → Standard times above
│       ├─► Paper bag → -1-2 days
│       └─► Exposed to air → -2-3 days
│
├─► PASTRIES/SWEET BAKED GOODS
│   │
│   ├─► What type?
│   │   ├─► High moisture (cream-filled, custard)
│   │   │   ├─► Refrigerated → 1-2 days
│   │   │   └─► Never room temp >2 hours (food safety)
│   │   │
│   │   ├─► Medium moisture (muffins, cake)
│   │   │   ├─► Room temp, covered → 2-4 days
│   │   │   ├─► Refrigerated → 5-7 days
│   │   │   └─► Frozen → 2-4 months
│   │   │
│   │   └─► Low moisture (cookies, biscotti)
│   │       ├─► Room temp, airtight → 1-2 weeks
│   │       ├─► Refrigerated → 2-3 weeks
│   │       └─► Frozen → 6-8 months
│   │
│   └─► Frosting/icing present?
│       ├─► Cream cheese frosting → Refrigerate, 3-5 days
│       ├─► Buttercream → Room temp OK, 2-3 days
│       └─► Fondant → Room temp, 1 week+
│
├─► PIZZA/FLATBREADS
│   │
│   ├─► Leftover pizza
│   │   ├─► Refrigerated → 3-4 days
│   │   └─► Frozen → 1-2 months
│   │
│   └─► Fresh dough
│       ├─► Refrigerated → 2-4 days
│       └─► Frozen → 3-4 months
│
└─► CRACKERS/DRY GOODS
    │
    ├─► Unopened → 6-12 months (check date)
    └─► Opened
        ├─► Airtight container → 3-6 months
        └─► Loosely covered → 1-2 months (goes stale)

QUALITY INDICATORS:
- Mold (any color) → DISCARD entire item
- Stale but no mold → Safe, quality reduced
- Hard/dry → Can refresh by heating
- Off smell → DISCARD
```

## Tree E: Canned/Jarred Goods

```
START: What's the acidity level?
│
├─► HIGH ACID (pH < 4.6)
│   Examples: Tomatoes, fruits, pickles, sauerkraut
│   │
│   ├─► Container condition?
│   │   ├─► Perfect condition
│   │   │   ├─► Unopened → 12-18 months (quality best)
│   │   │   ├─► Past date 1 year → Usually safe, check quality
│   │   │   └─► Past date 2+ years → Quality poor, safe if no damage
│   │   │
│   │   ├─► Dented but no rust
│   │   │   └─► Use within stated date
│   │   │
│   │   └─► Any of these → DISCARD IMMEDIATELY:
│   │       - Bulging
│   │       - Severe rust
│   │       - Leaking
│   │       - Spurts when opened
│   │       - Strong off odor
│   │
│   └─► After opening?
│       ├─► Transfer to glass/plastic container
│       ├─► Refrigerate
│       └─► Use within 5-7 days
│
├─► LOW ACID (pH > 4.6)
│   Examples: Meat, poultry, fish, vegetables
│   │
│   ├─► Container condition?
│   │   ├─► Perfect condition
│   │   │   ├─► Unopened → 2-5 years (quality best)
│   │   │   ├─► Past date 1 year → Usually safe if stored properly
│   │   │   └─► Past date 2+ years → Quality reduced, likely safe
│   │   │
│   │   └─► ANY damage → EXTREME CAUTION
│   │       ├─► Botulism risk in low-acid foods
│   │       ├─► When in doubt → THROW IT OUT
│   │       └─► Never taste to test
│   │
│   └─► After opening?
│       ├─► Transfer to non-metal container
│       ├─► Refrigerate immediately
│       └─► Use within 3-4 days
│
└─► JARRED FOODS (jams, pickles, condiments)
    │
    ├─► Unopened?
    │   ├─► Jams/jellies (high sugar) → 1-2 years
    │   ├─► Pickles (vinegar) → 1-2 years
    │   ├─► Condiments → Check specific type
    │   └─► Past date → Usually safe if sealed properly
    │
    └─► After opening?
        ├─► Jams/jellies → 6-12 months refrigerated
        ├─► Pickles → 3-4 months refrigerated
        ├─► Ketchup → 6 months refrigerated
        ├─► Mustard → 12 months refrigerated
        ├─► Mayonnaise → 2-3 months refrigerated
        └─► Salsa → 1-2 weeks refrigerated

CRITICAL: BOTULISM WARNING
Low-acid foods + damaged cans = DEADLY RISK
Never taste suspicious canned food
Symptoms appear 18-36 hours after consumption
Even tiny amounts can be fatal
```

## Tree F: Dried Foods

```
START: What moisture level?
│
├─► VERY DRY (<5% moisture)
│   Examples: Crackers, cookies, chips, cereal
│   │
│   ├─► Unopened?
│   │   ├─► Commercial packaging → 6-12 months
│   │   └─► Past date → Usually safe, may be stale
│   │
│   └─► Opened?
│       ├─► Airtight container?
│       │   ├─► YES → 3-6 months
│       │   └─► NO → 1-2 months (goes stale)
│       │
│       └─► Storage location?
│           ├─► Cool, dry pantry → Maximum life
│           ├─► Humid environment → -50% life
│           └─► Near heat source → -50% life
│
├─► MODERATELY DRY (5-15% moisture)
│   Examples: Dried fruits, jerky, dried vegetables
│   │
│   ├─► Commercial dried fruit
│   │   ├─► Unopened → 6-12 months
│   │   ├─► Opened, airtight → 3-6 months
│   │   └─► Refrigerated → 9-12 months
│   │
│   ├─► Jerky
│   │   ├─► Commercial, unopened → 12 months
│   │   ├─► Commercial, opened → 1-2 months
│   │   └─► Homemade → 1-2 months (refrigerate)
│   │
│   └─► Dried vegetables
│       ├─► Unopened → 1-2 years
│       └─► Opened, airtight → 6-12 months
│
├─► GRAINS & LEGUMES
│   │
│   ├─► WHITE RICE
│   │   ├─► Indefinite at room temp (if dry)
│   │   └─► Cooked → 4-6 days refrigerated
│   │
│   ├─► BROWN RICE / WHOLE GRAINS
│   │   ├─► Room temp → 6 months
│   │   ├─► Refrigerated → 1 year
│   │   ├─► Frozen → 18 months
│   │   └─► Oils in bran = shorter life
│   │
│   ├─► PASTA (dry)
│   │   ├─► White pasta → 2 years
│   │   ├─► Whole wheat pasta → 1 year
│   │   └─► Fresh pasta → 2-3 days refrigerated
│   │
│   ├─► FLOUR
│   │   ├─► White flour
│   │   │   ├─► Pantry → 1 year
│   │   │   ├─► Refrigerator → 2 years
│   │   │   └─► Freezer → Indefinite
│   │   │
│   │   └─► Whole wheat flour
│   │       ├─► Pantry → 3 months
│   │       ├─► Refrigerator → 6 months
│   │       └─► Freezer → 1 year
│   │
│   └─► DRIED BEANS/LEGUMES
│       ├─► Indefinite (quality declines after 2 years)
│       ├─► Older beans = longer cooking time
│       └─► Cooked → 3-4 days refrigerated
│
└─► NUTS & SEEDS
    │
    ├─► In shell → 6-12 months (room temp)
    ├─► Shelled
    │   ├─► Room temp → 2-3 months
    │   ├─► Refrigerated → 4-6 months
    │   └─► Frozen → 12 months
    │
    └─► Type matters:
        ├─► High fat nuts (walnuts, pecans) → Shorter life
        └─► Lower fat nuts (almonds) → Longer life

QUALITY CHECKS:
- Rancid smell (oil oxidation) → DISCARD
- Insects/webbing → DISCARD
- Mold → DISCARD
- Clumping (moisture absorption) → Usually safe, quality poor
```

## Tree G: Frozen Foods

```
START: Was it frozen properly from start?
│
├─► NO (frozen after time at room temp)
│   └─► Maximum safety guideline
│       ├─► Frozen within 2 hours of cooking → Safe
│       └─► Left out >2 hours → Don't freeze, DISCARD
│
└─► YES (properly frozen from fresh or after cooking)
    │
    ├─► What category?
    │   ├─► MEAT/POULTRY
    │   │   ├─► Ground meat → 3-4 months (quality)
    │   │   ├─► Steaks/roasts → 6-12 months
    │   │   ├─► Chicken pieces → 9 months
    │   │   ├─► Whole chicken → 12 months
    │   │   └─► Cooked meat dishes → 2-3 months
    │   │
    │   ├─► SEAFOOD
    │   │   ├─► Lean fish (cod, tilapia) → 6-8 months
    │   │   ├─► Fatty fish (salmon, tuna) → 2-3 months
    │   │   └─► Shellfish → 3-6 months
    │   │
    │   ├─► FRUITS/VEGETABLES
    │   │   ├─► Vegetables (blanched) → 8-12 months
    │   │   ├─► Fruits → 8-12 months
    │   │   └─► Unblanched vegetables → 2-3 months
    │   │
    │   ├─► DAIRY
    │   │   ├─► Butter → 6-9 months
    │   │   ├─► Ice cream → 2-3 months (quality)
    │   │   └─► Cheese → 6 months (texture changes)
    │   │
    │   ├─► BAKED GOODS
    │   │   ├─► Bread → 3-6 months
    │   │   ├─► Cookie dough → 3-6 months
    │   │   └─► Cakes/cookies (baked) → 2-4 months
    │   │
    │   └─► PREPARED MEALS
    │       ├─► Homemade → 2-3 months
    │       ├─► Commercial frozen meals → 3-6 months
    │       └─► Pizza → 1-2 months
    │
    ├─► Freezer temperature?
    │   ├─► 0°F (-18°C) or below → Standard times above
    │   ├─► 10-20°F (-12 to -7°C) → Reduce times by 50%
    │   └─► >20°F (>-7°C) → Not safe for long-term storage
    │
    ├─► Packaging quality?
    │   ├─► Vacuum sealed → +50% to standard times
    │   ├─► Tight plastic wrap → Standard times
    │   ├─► Freezer bags, air removed → Standard times
    │   └─► Poor packaging → -50% to times
    │
    └─► Quality indicators?
        ├─► Ice crystals inside package?
        │   └─► Indicates temperature fluctuation, use soon
        │
        ├─► Freezer burn (white/grey dry spots)?
        │   ├─► Safe to eat
        │   ├─► Quality reduced (tough, dry texture)
        │   └─► Can cut off affected areas
        │
        └─► Color changes?
            ├─► Meat browning/greying → Quality issue, safe
            └─► Vegetables dull → Quality issue, safe

THAWING GUIDELINES:
Safe methods:
1. Refrigerator (24-48 hours)
2. Cold water (change every 30 min)
3. Microwave (cook immediately)

Never:
- Room temperature thawing (bacteria growth)
- Hot water thawing (uneven, bacteria risk)

Refreezing:
- Thawed in refrigerator → Can refreeze (quality loss)
- Thawed other ways → Cook first, then can refreeze
```

## Quick Decision Guide

### "Is This Still Good?" Emergency Flowchart

```
START: Is there visible mold?
│
├─► YES
│   ├─► Hard cheese or firm produce → Cut 1 inch around, rest OK
│   └─► Any other food → DISCARD
│
└─► NO
    │
    ├─► Does it smell off/sour/rancid?
    │   └─► YES → DISCARD
    │
    ├─► Is the texture slimy, mushy, or wrong?
    │   └─► YES → DISCARD
    │
    ├─► Has it been at room temp >2 hours (>1 hour if >90°F)?
    │   └─► YES and it's perishable → DISCARD
    │
    ├─► Is it a damaged/bulging can?
    │   └─► YES → DISCARD (botulism risk)
    │
    ├─► Is it past the date but looks/smells fine?
    │   ├─► "Use By" date → Use caution, probably OK if stored properly
    │   ├─► "Sell By" date → Usually OK for days/weeks after
    │   └─► "Best By" date → Quality date only, usually safe after
    │
    └─► When in doubt → THROW IT OUT
```

## Date Label Decoder

```
"SELL BY" → For retailers, not consumers
           → Safe to use 1-7 days after for most items
           → Example: Milk "sell by Jan 15" is usually good until Jan 22

"BEST BY" / "BEST BEFORE" → Quality date, not safety
                          → Food may lose quality but is usually safe
                          → Example: Crackers "best by June 1" are safe but may be stale

"USE BY" → Last day for peak quality
         → Only safety date on infant formula
         → Other foods usually safe shortly after
         → Example: Yogurt "use by Jan 10" often good until Jan 17

"FREEZE BY" → Date to freeze for best quality
            → Can still freeze after, but quality may decline
            → Example: Chicken "freeze by Jan 5" → freeze ASAP for best results
```

## Special Considerations

### Temperature Fluctuations
```
Power outage occurred?

├─► Freezer
│   ├─► Still has ice crystals → Safe to refreeze
│   ├─► Thawed completely
│   │   ├─► Still cold (40°F or below) → Cook within 24 hours
│   │   └─► Warm → DISCARD
│   └─► Kept closed, full freezer → Safe 48 hours
│
└─► Refrigerator
    ├─► Kept closed → Safe 4 hours
    ├─► Still at 40°F or below after 4 hours → Safe
    └─► Warmed above 40°F → Check each item
```

### Food Left Out
```
How long at room temperature?

├─► <2 hours (< 1 hour if >90°F)
│   └─► Refrigerate immediately, safe
│
├─► 2-4 hours
│   ├─► High risk foods (meat, dairy) → DISCARD
│   └─► Low risk foods (bread, dry goods) → Probably OK
│
└─► >4 hours
    └─► DISCARD all perishable foods
```
